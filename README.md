💈 Barbershop Booking System:

A modern full-stack appointment booking platform that enables customers to book barber appointments online with ease. Features real-time scheduling, WhatsApp notifications, and a responsive user experience.

🚀 Tech Stack:

Next.js • React • MongoDB • Tailwind CSS • Vercel

✨ Features:

* 📅 Online Appointment Booking
* ⏰ Real-Time Schedule Management
* 📲 WhatsApp Notifications
* 💰 Service & Pricing Management
* 📱 Responsive Design
* 🔒 Secure & Fast Booking Experience
